# Runnable TypeScript lessons

Uruchamialne przyklady kodu do materialow z `docs/typescript/01...10`.

## Pliki

- `01-podstawy.ts` - typy podstawowe i funkcje
- `02-typy-w-praktyce.ts` - interfejsy, union type, `Partial<T>`
- `03-async-await.ts` - asynchronicznosc i `await`
- `04-patterny-testowe.ts` - wzorce z automatyzacji testow
- `05-sciagawka.ts` - utility types (`Pick`, `Record`, `Partial`)
- `06-zmienne-warunki-petle.ts` - warunki i petle w praktyce
- `07-obiekty-tablice-funkcje.ts` - obiekty, tablice i funkcje
- `08-bledy-try-catch.ts` - bezpieczna obsluga bledow
- `09-praca-z-api-json.ts` - typowanie odpowiedzi API
- `10-playwright-od-zera.ts` - mini flow testowy

## Uruchamianie

```bash
npm run learn:01
npm run learn:02
npm run learn:03
npm run learn:04
npm run learn:05
npm run learn:06
npm run learn:07
npm run learn:08
npm run learn:09
npm run learn:10
```

Lub wszystko naraz:

```bash
npm run learn:all
```
