# Runnable TypeScript lessons

Uruchamialne przyklady kodu do materialow z `docs/typescript/01...05`.

## Pliki

- `01-podstawy.ts` - typy podstawowe i funkcje
- `02-typy-w-praktyce.ts` - interfejsy, union type, `Partial<T>`
- `03-async-await.ts` - asynchronicznosc i `await`
- `04-patterny-testowe.ts` - wzorce z automatyzacji testow
- `05-sciagawka.ts` - utility types (`Pick`, `Record`, `Partial`)

## Uruchamianie

```bash
npm run learn:01
npm run learn:02
npm run learn:03
npm run learn:04
npm run learn:05
```

Lub wszystko naraz:

```bash
npm run learn:all
```
